# FormTranslator 安装和使用指南

## 📦 安装步骤

### 开发者模式安装（推荐用于测试）

1. **打开Chrome浏览器**
   - 确保使用Chrome浏览器版本88或更高版本

2. **进入扩展程序管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或者点击右上角三点菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击开关启用开发者模式

4. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择FormTranslator项目文件夹
   - 点击"选择文件夹"

5. **验证安装**
   - 插件应该出现在扩展程序列表中
   - 浏览器工具栏应该显示FormTranslator图标

## ⚙️ 初始配置

### 1. 打开设置页面
- 点击插件图标，然后点击"⚙️ 设置"
- 或者在扩展程序管理页面点击"详细信息" → "扩展程序选项"

### 2. 选择翻译引擎

#### Google 翻译（推荐）
- **免费版本**：无需API密钥，但有使用限制
- **付费版本**：需要Google Cloud Translation API密钥
  1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
  2. 创建项目或选择现有项目
  3. 启用Translation API
  4. 创建API密钥
  5. 将API密钥粘贴到设置页面

#### DeepL 翻译
- **需要API密钥**（有免费额度）
  1. 访问 [DeepL API](https://www.deepl.com/pro-api)
  2. 注册账户
  3. 获取免费API密钥
  4. 将API密钥粘贴到设置页面

### 3. 设置目标语言
- 在语言网格中选择您希望翻译到的目标语言
- 默认为中文（简体）

### 4. 测试配置
- 点击"测试翻译"按钮验证设置是否正确
- 如果测试成功，会显示翻译示例

## 🚀 使用方法

### 方法一：点击插件图标
1. 在包含表单的网页上点击FormTranslator图标
2. 在弹出窗口中点击"🚀 翻译表单"按钮
3. 等待翻译完成

### 方法二：使用快捷键
1. 在包含表单的网页上按下快捷键：
   - Windows/Linux: `Ctrl + Shift + T`
   - Mac: `Command + Shift + T`
2. 插件会自动开始翻译表单内容

### 支持的表单元素
- 文本输入框 (`<input type="text">`)
- 文本区域 (`<textarea>`)
- 邮箱输入框 (`<input type="email">`)
- 电话输入框 (`<input type="tel">`)
- 搜索框 (`<input type="search">`)
- URL输入框 (`<input type="url">`)
- 数字输入框 (`<input type="number">`)
- 下拉选择框 (`<select>`)

## 🧪 测试插件

### 使用测试页面
1. 在浏览器中打开项目文件夹中的 `test.html` 文件
2. 双击页面标题自动填充测试数据
3. 使用插件翻译表单内容

### 测试步骤
1. **填充测试数据**：在表单中输入英文内容
2. **触发翻译**：使用插件图标或快捷键
3. **验证结果**：检查表单内容是否被正确翻译
4. **测试不同场景**：
   - 空表单（应该显示"未检测到表单元素"）
   - 部分填充的表单
   - 不同类型的表单元素

## 🔧 故障排除

### 常见问题

#### 1. 插件图标不显示
- 检查是否正确加载了插件
- 刷新扩展程序页面
- 重新加载插件

#### 2. 翻译失败
- 检查网络连接
- 验证API密钥是否正确
- 查看浏览器控制台错误信息
- 尝试切换翻译引擎

#### 3. 快捷键不工作
- 检查快捷键设置是否启用
- 确认没有其他扩展程序使用相同快捷键
- 在chrome://extensions/shortcuts 中检查快捷键配置

#### 4. 某些表单元素未被翻译
- 确认元素类型是否受支持
- 检查元素是否有内容
- 查看浏览器控制台是否有错误信息

### 调试模式
1. 打开浏览器开发者工具 (F12)
2. 查看Console标签页的错误信息
3. 在Network标签页检查API请求状态

## 📝 开发说明

### 项目结构
```
FormTranslator/
├── manifest.json          # 插件配置文件
├── popup.html/js          # 弹出界面
├── content.js             # 内容脚本
├── background.js          # 后台脚本
├── options.html/js        # 设置页面
├── icons/                 # 图标文件
├── test.html             # 测试页面
└── README.md             # 项目说明
```

### 修改和自定义
- **添加新的翻译引擎**：修改 `content.js` 和 `background.js`
- **支持新的表单元素**：更新 `content.js` 中的选择器
- **修改界面样式**：编辑对应的HTML和CSS文件
- **添加新语言**：更新 `options.js` 中的语言列表

## 🔒 隐私和安全

- 插件只在用户主动触发时才会翻译内容
- API密钥存储在浏览器本地，不会上传到服务器
- 翻译请求直接发送到选定的翻译服务提供商
- 不会收集或存储用户的表单数据
